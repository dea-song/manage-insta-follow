# Insta Tools

인스타그램 팔로워 관리를 도와주는 간단한 웹 도구입니다.

## 기능

### 팔로워 Diff 추출기
- 두 텍스트를 비교하여 첫 번째 기준으로 두 번째에 없는 항목만 추출
- 개행 또는 쉼표로 구분, 순서 무관

### 팔로우 목록
- 아이디 목록을 입력하면 팔로우 링크를 목록으로 정리
- 팔로우 버튼 클릭 시 인스타그램 프로필이 팝업으로 열림
- 완료한 항목은 자동으로 체크 표시

## GitHub Pages 배포 방법

1. 이 저장소를 GitHub에 push
2. 저장소 Settings → Pages
3. Source: `Deploy from a branch`
4. Branch: `main` / `(root)` 선택 후 Save
5. 잠시 후 `https://{username}.github.io/{repo-name}/` 으로 접속

## 로컬 실행

별도 설치 없이 `index.html`을 브라우저에서 바로 열면 됩니다.
